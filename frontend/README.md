# Frontend Documentation

## Overview
Static web interface for DocuMind Intelligence hosted on Amazon S3.

## Files
- index.html - Main application interface

## Features
- Drag and drop file upload
- Real-time document analysis
- Beautiful gradient UI
- Responsive design

## Setup
1. Update API_ENDPOINT in index.html with your API Gateway URL
2. Upload to S3 bucket
3. Enable static website hosting
4. Set index.html as index document

## S3 Bucket Configuration
- Enable static website hosting
- Add bucket policy for public read access
- Enable CORS if needed

## Local Testing
Simply open index.html in a browser after updating the API endpoint.
